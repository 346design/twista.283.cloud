================================================================

**A forever evolving, sophisticated microblogging platform.**

[Groundpolis](https://social.xeltica.work") は [Misskey](https://github.com/xeltica/groundpolis)をベースとしたマイクロブログサービスです。

## 変更点

- アドブロック対策を削除（うるせえしな）
- 削除して編集機能 (そろそろマージされる)
- ブランド名の変更 (Misskey -> Groundpolis)
- 藍 -> シトリン
